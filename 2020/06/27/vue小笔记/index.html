vue初学时的笔记

vue属性绑定时，如果是传入’{}’,那括号内的是对象

  • 譬如class、style属性绑定
注意是对象，用’,’隔开，另外’font-size’写成了’fontSize’ 且键值对的写法，value要用字符串形式 color:’#f40’

引号与反引号

以上template使用的是引号，且只能写在一行，因为引号是无法响应换行符的，会直接在换行符处截止。
如果使用引号而且想换行，可以在需要换行的末尾使用转义字符’'。。或者使用’+’拼凑 。。或者改用反引号 如下：

component是使用的反引号，可以随意换行，暂不能拼接

Vue.extend和component都可以随意使用引号或者反引号
咱总结如下：其实可以随意使用引号或者是反引号，选用引号的问题就是如下：
template使用的是引号，这就意味着不能处理换行符，引号到换行符就截止了,即只能写在一行。如果想写在多行，有三种方法，其一：在需换行处使用转义字符’',把换行符转义；其二：在需要换行处,使用’+’字符串拼接起来；其三：不用引号，改用反引号，就可以换行，但是不能拼接；

文件路径写法

‘./‘ 当前文件夹
‘../‘ 父级文件夹
‘@/‘ ‘@’是webpack定义的 可以在’webpack.base.conf.js’中查看‘@’的代表路径是什么 一般vue-cli中‘@’指向的是’src’文件夹
dist目录 distrubution发布
npm install axios –save 后面的’–save’是安装时在package.json注册就可以在生产d环境用了

意思就是，生产环境时(需要放在服务器的)就需要’–save’
